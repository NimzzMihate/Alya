## 🚀 Quick Start

### Prerequisites

```bash
node -v  # v18 atau lebih baru
npm -v   # v8 atau lebih baru
```

### Installation

```bash
# Clone repository
git clone https://github.com/NimzzMihate/Alya.git
cd Alya

# Install dependencies
npm install

# Konfigurasi
cp x-system/config-set.json.example x-system/config-set.json
# Edit config-set.json sesuai kebutuhan

# Jalankan bot
node index.js
```

### Konfigurasi

Edit file `x-system/config-set.json`:

```json
{
  "owner": "628xxxxxxxxxx",
  "botname": "Alya MD",
  "githubToken": "your_github_token",
  "githubDataRepo": "username/alya-data",
  "websiteUrl": "https://your-website.vercel.app"
}
```

## 📋 Beberapa Command

```
.menu          → Lihat semua command
.daftar        → Daftar akun RPG
.adventure     → Mulai petualangan RPG
.mancing       → Mini game mancing
.snippet save  → Simpan code snippet
.ai            → Chat dengan AI
.yt            → Download dari YouTube
.sticker       → Buat sticker dari gambar
```
